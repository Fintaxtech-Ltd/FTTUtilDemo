Skills for the demo app module.

## Skill: Add Utility Demo

Use when demonstrating an existing utility from `ftt-utils`.

### Generate

- Small demo usage
- Sample inputs/outputs
- Optional simple UI if the app already uses UI

### Rules

- Import utility from `ftt-utils`.
- Do not duplicate utility implementation.
- Keep demo sample-only.
- Do not add business logic.
