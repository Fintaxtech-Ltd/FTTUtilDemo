# app/AGENTS.md

Instructions for the demo app module.

## Module Purpose

`app` is the demo/sample application shell for the FTT utilities library.

## Responsibilities

- Demonstrate utility usage.
- Provide manual verification examples.
- Host simple sample screens only if needed.

## Forbidden

- Do not add reusable utility implementations here.
- Do not add publishing logic here.
- Do not make utilities depend on this app.
- Do not add business/domain architecture here.

## Rules

- Keep the app small.
- Use it only to demonstrate `ftt-utils`.
- Demo code may be simple and sample-only.
