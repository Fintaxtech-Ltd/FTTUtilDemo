# SKILLS.md

Skill index for the FTT Utilities Library repository.

Use the nearest module SKILLS.md for detailed instructions.

---

## Skill: Create Utility

Use when adding a reusable utility class/object/function to `ftt-utils`.

Detailed instructions:
- `ftt-utils/SKILLS.md`

Expected output:
- Utility file
- Matching unit test file
- KDoc when needed

---

## Skill: Add Tests for Existing Utility

Use when a utility exists but does not have complete method coverage.

Detailed instructions:
- `ftt-utils/SKILLS.md`

Expected output:
- Unit test file or added test methods
- Tests for every public method and edge case

---

## Skill: Refactor Utility API

Use when improving naming, purity, safety, or public API consistency.

Detailed instructions:
- `ftt-utils/SKILLS.md`

Expected output:
- Refactored utility
- Updated tests
- Migration notes if API changes
